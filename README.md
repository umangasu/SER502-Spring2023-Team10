# SER502-Spring2023-Team10

# Prerequisites
  System - MacOS.
  
  Tools - Python3
  
# What to install
  Install:

  Python3

  Prolog (swipl)
  
# Instructions to Build & Run
In the main folder 'SER502-Spring2023-Team10' run the following command
```bash
python runeva.py <filename>
```
Add your .eva files to '/data' folder to run.
